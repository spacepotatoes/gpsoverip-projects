# GPSauge MI6 – Fuhrpark/Telematik-Slide

Aufgabe: eine Instagram/Social-Media-Infografik ("1. Fahrzeugdaten erfassen") zur
Telematik-Hardware GPSauge MI6 wird als vollständig editierbares Figma-Layout
nachgebaut (alle Elemente als eigene Ebenen, nicht als ein flaches Bild).

## Figma-Datei

https://www.figma.com/design/U6lzjE0TSKUQ8H19G0ofNJ

Seite "1 – Fahrzeugdaten erfassen", Frame 1080×1350.

## Stand (21.06.2026)

Fertig als eigene Ebenen gebaut:
- Orangener Kreis mit "1." (Ellipse + Text getrennt)
- Headline "Fahrzeugdaten erfassen" (Inter Black)
- Weiße Hinweisbox mit orangenem Rahmen, kursiver Text "Telematik-Hardware
  GPSauge MI6 sitzt direkt im Fahrzeug" (Inter Bold Italic)
- Checkliste: 4 Zeilen, je orangener Häkchen-Kreis + Label (Inter Semi Bold) —
  GPS-Ortung / Fahrzeugdaten über CAN/FMS / Über 100 Telemetriewerte /
  Tacho-Remote-Download
- Unterer oranger Balken mit Fließtext "Daten werden automatisch an das
  Fuhrparkportal übertragen"

Offen / Platzhalter (beschriftete Rahmen in Figma, noch ohne echtes Bild):
- Hintergrundfoto LKW-Flotte (oberer Bereich, Frame "PLATZHALTER Foto –
  Hintergrund (LKW-Flotte)")
- Produktfoto GPSauge MI6 PRO, 2 Geräte freigestellt (Frame "PLATZHALTER Foto
  – GPSauge MI6 PRO")

Grund für die Platzhalter: Nur das fertig zusammengesetzte Referenzbild lag
vor, keine separaten Rohdateien/Freisteller — die müssen noch eingesetzt
werden (in Figma direkt per Drag&Drop auf die Platzhalter-Frames).

## Zugriff

- Figma-Account: Giusi (Pro-Plan, Dev-Seat) — Login reicht, kein weiteres
  Setup nötig.
- Figma MCP Tools (`use_figma` etc.) sind nur in einer Claude-Code-Session
  nutzbar, die mit dem Figma-Account authentifiziert ist.

  einsetzen
- Feinschliff: Farben/Schrift/Abstände gegen das Original-Referenzbild
  abgleichen, falls gewünscht
